# Images Folder

Place your company logo here:

- **logo.png** - Main company logo (recommended: PNG with transparent background)
- **logo.svg** - Vector logo (optional, better for scaling)

## Logo Requirements:

- **Format**: PNG (with transparency) or SVG
- **Size**: Recommended 200-300px width
- **Aspect Ratio**: Maintain original proportions
- **Background**: Transparent preferred

## File Name:
The logo should be named: **logo.png**

The website will automatically use this logo. If the logo file is not found, it will display a text-based fallback with "PAT IOT SOLUTIONS".
