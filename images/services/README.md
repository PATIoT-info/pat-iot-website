# Services Images Folder

Place your service images here with the following filenames:

## Required Service Images:

1. **motorised-curtains.jpg** - Motorised Curtains service
2. **smart-door-locks.jpg** - Smart Door Locks service
3. **ir-blasters.jpg** - IR Blasters service
4. **smart-door-bell.jpg** - Smart Door Bell service
5. **motion-sensors.jpg** - Motion Sensors service
6. **security-systems.jpg** - Security Systems service
7. **climate-control.jpg** - Climate Control service
8. **voice-control.jpg** - Voice Control service

## Image Requirements:

- **Format**: JPG or PNG
- **Recommended Size**: 400x300px (or similar aspect ratio)
- **Aspect Ratio**: 4:3 (landscape)
- **File Size**: Optimized for web (under 150KB per image recommended)
- **Quality**: High quality, clear product/service photos

## Notes:

- Images will display in the Services & Solutions section
- If an image is missing, the service card will fall back to showing an icon
- Images should show the actual products/services clearly
- All services now use images instead of icons
