## Image Requirements:

- **Format**: JPG or PNG
- **Recommended Size**: 700x500px (or similar aspect ratio)
- **Aspect Ratio**: 1.4:1 (landscape)
- **File Size**: Optimized for web (under 200KB per image recommended)
- **Quality**: High quality, clear product photos

## Notes:

- Images will automatically display with product labels and captions
- If an image is missing, a placeholder will be shown
- Images should show the product clearly and professionally
